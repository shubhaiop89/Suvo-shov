##### ğŸ”¹Notes

Supported operation types:
â€¢ set - Set key-value pairs
â€¢ get - Read values (for read-your-writes consistency)
â€¢ add - Add items to collections
â€¢ update - Update collection items by ID
â€¢ remove - Remove collection items by ID
â€¢ forget - Delete keys
â€¢ clear - Clear entire collections

âš ï¸ IMPORTANT: All operations in a batch are executed atomically. If any operation fails, the entire batch is rolled back and no changes are made.
